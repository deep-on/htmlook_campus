> HTMLook Pro をインストールして初めて起動します。約 3 分。

このステップが終わると、アプリが Dock に常駐し、空のワークスペースが 1 つ
開き、ファイルを 1 つドラッグ&ドロップしてビューワーに表示されるところまで
確認できます。

## 1.1 · ダウンロードとインストール

1. [htmlook.app](https://htmlook.app) → **Download for macOS** をクリック。
   .dmg を取得します。(Windows / iPad は後続リリース。)
2. .dmg をダブルクリック → HTMLook Pro を Applications にドラッグ。
3. Launchpad から HTMLook Pro を起動。初回起動時に Gatekeeper が確認を
   求めたら *開く*。

> **14 日間の無料トライアル**、全機能が有効。事前の決済情報入力なし。
> トークンは BYOM なのでマージン 0 — ご自身のベンダーキーで直接呼び出し
> ([Step 3](../03-byom-setup/) 参照)。

## 1.2 · 初回起動 — 空のワークスペース

アプリが開くと 3 つの領域が見えます:

- 左: サイドバー (今は空 · *Open Folder* ボタン)
- 中央: ビューワーキャンバス (今は *Drop a file here* プレースホルダ)
- 右: パネル領域 (Settings / Tutorials / AI / Terminal · 切り替え可)

## 1.3 · 最初のファイル — drop & open

Finder から好きな HTML ファイルを 1 つ、中央のキャンバスに
ドラッグして落としてください。
手近にない場合は
[`../05-command/personas/01-hf-claude/landing.html`](../05-command/personas/01-hf-claude/landing.html)
がすぐそばにあります。

> 動画で確認: [BASIC #1 · Drop. Open. Done.](../videos/features/01-basic-01-drop-open.mp4) (30 秒)

## 1.4 · フォルダ単位で開く (workspace)

アプリメニューの *File → Open Folder*、またはサイドバーの *Open Folder*
ボタン。フォルダを 1 つ選ぶと、サイドバーがそのフォルダのツリーに
切り替わります。

キャンパスのステップフォルダのどれを選んでも構いません。コンテンツが
最も多い場所:
[`../05-command/personas/`](../05-command/personas/) (26 ペルソナフォルダ)。

> 動画: [BASIC #2 · Folder ready.](../videos/features/02-basic-02-workspace.mp4) (30 秒)

## 1.5 · Tutorials タブ (インタラクティブガイド)

アプリメニュー *Help → Interactive Tutorials…* を開きます。動画 17 本
それぞれに対応する spotlight + tooltip ガイド (4 ステップ) が組み込まれて
います。動画が *what (何ができるか)* を見せるなら、Tutorials は
*how (どうやるか)* を実際の UI 上で案内します。

## ✅ Step 1 チェックポイント

- HTMLook Pro がインストールされ、Dock から起動できる。
- ファイルをドラッグ&ドロップ → ビューワーに表示される。
- フォルダ 1 つを *Open Folder* で開いたら、サイドバーがツリーで埋まった。

3 つ揃ったら → [Step 2 · 最初のファイル (HTML / MD / PDF / 動画)](../02-first-file/)
