> Installez HTMLook Pro et lancez-le pour la première fois. ~ 3 min.

À la fin de cette étape : l'app est dans votre dock, un workspace vide est
ouvert, et vous avez glissé-déposé un fichier pour le voir s'afficher dans
le viewer.

## 1.1 · Téléchargement & installation

1. Allez sur [htmlook.app](https://htmlook.app) → cliquez **Download for macOS**.
   Récupérez le .dmg. (Windows / iPad sortent plus tard.)
2. Double-clic sur le .dmg → glissez HTMLook Pro dans Applications.
3. Lancez HTMLook Pro depuis Launchpad. Si Gatekeeper demande au premier
   lancement : *Ouvrir*.

> **Essai gratuit de 14 jours** avec toutes les fonctionnalités. Pas
> d'informations de paiement à l'avance.
> Les tokens passent par BYOM — zéro marge, appelés directement avec votre
> propre clé fournisseur (voir [Step 3](../03-byom-setup/)).

## 1.2 · Premier lancement — workspace vide

À l'ouverture, vous voyez trois zones :

- À gauche : sidebar (vide pour l'instant · bouton *Open Folder*)
- Au centre : canvas du viewer (placeholder *Drop a file here*)
- À droite : zone de panneaux (Settings / Tutorials / AI / Terminal ·
  basculables)

## 1.3 · Premier fichier — drop & open

Glissez n'importe quel fichier HTML depuis le Finder vers le canvas au
centre. Si vous n'en avez pas sous la main,
[`../05-command/personas/01-hf-claude/landing.html`](../05-command/personas/01-hf-claude/landing.html)
est juste à côté.

> En vidéo : [BASIC #1 · Drop. Open. Done.](../videos/features/01-basic-01-drop-open.mp4) (30 s)

## 1.4 · Ouvrir un dossier (workspace)

Menu de l'app *File → Open Folder*, ou bouton *Open Folder* dans la sidebar.
Choisissez un dossier, la sidebar bascule sur son arborescence.

N'importe quel dossier d'étape du campus fait l'affaire. Celui avec le plus
de contenu :
[`../05-command/personas/`](../05-command/personas/) (26 dossiers de
persona).

> Vidéo : [BASIC #2 · Folder ready.](../videos/features/02-basic-02-workspace.mp4) (30 s)

## 1.5 · Onglet Tutorials (guide interactif)

Ouvrez dans le menu de l'app *Help → Interactive Tutorials…*. Chaque
vidéo (17 au total) a un guide spotlight + tooltip (4 étapes) intégré. Là
où les vidéos montrent *quoi* est possible, les Tutorials montrent
*comment* le faire sur l'UI réelle.

## ✅ Checkpoint Step 1

- HTMLook Pro est installé et se lance depuis le dock.
- Un fichier glissé-déposé apparaît dans le viewer.
- *Open Folder* sur un dossier réel a rempli la sidebar avec son
  arborescence.

Les trois ? → [Step 2 · Premier fichier (HTML / MD / PDF / vidéo)](../02-first-file/)
