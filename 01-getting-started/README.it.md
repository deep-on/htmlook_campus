> Installa HTMLook Pro e avvialo per la prima volta. ~ 3 min.

Alla fine di questo step: l'app è nel dock, un workspace vuoto è aperto,
e hai trascinato un file per vederlo comparire nel viewer.

## 1.1 · Download & installazione

1. Vai su [htmlook.app](https://htmlook.app) → clicca **Download for macOS**.
   Prendi il .dmg. (Windows / iPad arrivano dopo.)
2. Doppio clic sul .dmg → trascina HTMLook Pro in Applications.
3. Avvia HTMLook Pro dal Launchpad. Se al primo avvio Gatekeeper chiede,
   scegli *Apri*.

> **Prova gratuita di 14 giorni** con tutto sbloccato. Nessun dato di
> pagamento in anticipo.
> I token sono BYOM — margine zero, chiamati direttamente con la tua
> chiave del fornitore (vedi [Step 3](../03-byom-setup/)).

## 1.2 · Primo avvio — workspace vuoto

All'apertura vedi tre zone:

- A sinistra: sidebar (vuota per ora · pulsante *Open Folder*)
- Al centro: canvas del viewer (placeholder *Drop a file here*)
- A destra: area pannelli (Settings / Tutorials / AI / Terminal ·
  attivabili)

## 1.3 · Primo file — drop & open

Trascina un file HTML qualsiasi dal Finder al canvas centrale.
Se non ne hai uno a portata,
[`../05-command/personas/01-hf-claude/landing.html`](../05-command/personas/01-hf-claude/landing.html)
sta proprio lì accanto.

> Guardalo: [BASIC #1 · Drop. Open. Done.](../videos/features/01-basic-01-drop-open.mp4) (30 s)

## 1.4 · Apri una cartella (workspace)

Menu dell'app *File → Open Folder*, oppure pulsante *Open Folder* nella
sidebar. Scegli una cartella qualsiasi e la sidebar passa al suo albero.

Va bene qualsiasi cartella di step del campus. Quella con più contenuto:
[`../05-command/personas/`](../05-command/personas/) (26 cartelle di
persona).

> Video: [BASIC #2 · Folder ready.](../videos/features/02-basic-02-workspace.mp4) (30 s)

## 1.5 · Tab Tutorials (guida interattiva)

Dal menu dell'app, apri *Help → Interactive Tutorials…*. Ciascuno dei 17
video di feature ha una guida spotlight + tooltip (4 step) integrata.
Dove i video mostrano *cosa* è possibile, i Tutorials mostrano *come* si
fa sulla UI vera.

## ✅ Checkpoint Step 1

- HTMLook Pro è installato e si avvia dal dock.
- Un file trascinato compare nel viewer.
- *Open Folder* su una cartella vera ha riempito la sidebar con il suo
  albero.
