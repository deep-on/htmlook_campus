> Instale o HTMLook Pro e abra pela primeira vez. ~ 3 min.

No fim deste passo: o app fica no dock, um workspace vazio está aberto, e
você arrastou e soltou um arquivo para vê-lo aparecer no viewer.

## 1.1 · Download & instalação

1. Vá em [htmlook.app](https://htmlook.app) → clique **Download for macOS**.
   Pegue o .dmg. (Windows / iPad saem depois.)
2. Duplo clique no .dmg → arraste o HTMLook Pro para Applications.
3. Abra o HTMLook Pro pelo Launchpad. Se o Gatekeeper perguntar na
   primeira vez, escolha *Abrir*.

> **Trial gratuito de 14 dias** com tudo liberado. Sem dados de pagamento
> antecipados.
> Os tokens são BYOM — margem zero, chamados direto com a sua chave do
> fornecedor (ver [Step 3](../03-byom-setup/)).

## 1.2 · Primeira execução — workspace vazio

Quando o app abre, você vê três áreas:

- À esquerda: sidebar (vazia por enquanto · botão *Open Folder*)
- No centro: canvas do viewer (placeholder *Drop a file here*)
- À direita: área de painéis (Settings / Tutorials / AI / Terminal ·
  alternáveis)

## 1.3 · Primeiro arquivo — drop & open

Arraste qualquer arquivo HTML do Finder para o canvas no centro.
Se não tiver um à mão,
[`../05-command/personas/01-hf-claude/landing.html`](../05-command/personas/01-hf-claude/landing.html)
está logo ao lado.

> Veja em vídeo: [BASIC #1 · Drop. Open. Done.](../videos/features/01-basic-01-drop-open.mp4) (30 s)

## 1.4 · Abrir uma pasta (workspace)

Menu do app *File → Open Folder*, ou botão *Open Folder* na sidebar.
Escolha uma pasta e a sidebar troca para a árvore dela.

Qualquer pasta de passo do campus serve. A com mais conteúdo:
[`../05-command/personas/`](../05-command/personas/) (26 pastas de
persona).

> Vídeo: [BASIC #2 · Folder ready.](../videos/features/02-basic-02-workspace.mp4) (30 s)

## 1.5 · Aba Tutorials (guia interativo)

No menu do app, abra *Help → Interactive Tutorials…*. Cada um dos 17
vídeos de feature tem um guia spotlight + tooltip (4 passos) embutido.
Onde os vídeos mostram *o quê* é possível, os Tutorials mostram *como*
fazer na UI real.

## ✅ Checkpoint Step 1

- HTMLook Pro está instalado e abre pelo dock.
- Um arquivo arrastado aparece no viewer.
- *Open Folder* em uma pasta real preencheu a sidebar com a árvore dela.

Os três? → [Step 2 · Primeiro arquivo (HTML / MD / PDF / vídeo)](../02-first-file/)
