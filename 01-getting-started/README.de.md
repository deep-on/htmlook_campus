> Installiere HTMLook Pro und starte es zum ersten Mal. ~ 3 min.

Am Ende dieses Schritts liegt die App im Dock, ein leerer Workspace ist
geöffnet, und du hast per Drag & Drop eine Datei in den Viewer fallen
lassen.

## 1.1 · Download & Installation

1. Geh auf [htmlook.app](https://htmlook.app) → klicke **Download for macOS**.
   Hol dir die .dmg. (Windows / iPad folgen später.)
2. Doppelklick auf die .dmg → zieh HTMLook Pro in Applications.
3. Starte HTMLook Pro aus dem Launchpad. Wenn Gatekeeper beim ersten Start
   fragt: *Öffnen*.

> **14 Tage kostenlos** mit jedem Feature freigeschaltet. Keine
> Zahlungsinfos vorab.
> Tokens laufen via BYOM — null Aufschlag, direkt mit deinem eigenen
> Vendor-Key aufgerufen (siehe [Step 3](../03-byom-setup/)).

## 1.2 · Erster Start — leerer Workspace

Wenn die App öffnet, siehst du drei Bereiche:

- Links: Sidebar (jetzt leer · *Open Folder*-Button)
- Mitte: Viewer-Canvas (jetzt ein *Drop a file here*-Platzhalter)
- Rechts: Panel-Bereich (Settings / Tutorials / AI / Terminal · umschaltbar)

## 1.3 · Erste Datei — drop & open

Zieh eine beliebige HTML-Datei aus dem Finder in den Canvas in der Mitte.
Falls du keine zur Hand hast,
[`../05-command/personas/01-hf-claude/landing.html`](../05-command/personas/01-hf-claude/landing.html)
liegt direkt nebenan.

> Sieh's dir an: [BASIC #1 · Drop. Open. Done.](../videos/features/01-basic-01-drop-open.mp4) (30 s)

## 1.4 · Ordner öffnen (Workspace)

App-Menü *File → Open Folder*, oder den *Open Folder*-Button in der Sidebar.
Wähl einen beliebigen Ordner und die Sidebar wechselt in dessen Baumansicht.

Jeder Step-Ordner aus dem Campus funktioniert. Der Ordner mit dem meisten
Content:
[`../05-command/personas/`](../05-command/personas/) (26 Persona-Ordner).

> Video: [BASIC #2 · Folder ready.](../videos/features/02-basic-02-workspace.mp4) (30 s)

## 1.5 · Tutorials-Tab (interaktive Anleitung)

Öffne im App-Menü *Help → Interactive Tutorials…*. Zu jedem der 17
Feature-Videos gibt es einen Spotlight-+-Tooltip-Guide (4 Schritte). Wo
die Videos zeigen *was* möglich ist, zeigen die Tutorials *wie* es auf
der echten UI geht.

## ✅ Step 1 Checkpoint

- HTMLook Pro ist installiert und startet aus dem Dock.
- Eine per Drag & Drop abgelegte Datei erscheint im Viewer.
- *Open Folder* auf einen echten Ordner hat die Sidebar mit dem Baum gefüllt.

Alle drei? → [Step 2 · Erste Datei (HTML / MD / PDF / Video)](../02-first-file/)
