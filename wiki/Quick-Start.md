# Quick start

> 5 minutes from "I just opened HTMLook" to "I can do useful work in it."

This walkthrough assumes you've already installed HTMLook Pro ([Installation](Installation.md)) and the app is open.

## 1. Pick a workspace (30 s)

First-launch, HTMLook asks you to point at a folder. Anything works — `~/Notes`, your client folder, even an empty `~/Desktop/scratch`. That folder *is* your workspace from here on. (You can add more later; each workspace is independent.)

If you'd rather start from a curated example, pick **Open Sample Workspace** instead — HTMLook drops one onto your desk with a few example files to play with.

## 2. Open a file (15 s)

Single-click anything in the sidebar to open it in the centre. Double-click to open it as a new tab. The renderer changes per file type — Markdown gets the WYSIWYG editor, PDF gets PDF.js with highlighting, video / audio get a player with bookmarks.

## 3. Try the AI Assistant (1 min · Pro)

Hit ⌘I — the right pane opens. First time, a 5-card setup walks you through picking a provider (OpenAI / DeepSeek / Ollama / …), pasting your API key, and choosing a model. Then ask it something about the file you just opened:

> *"summarize this in 3 bullets"*

Watch it stream the answer. Cancel any time with ⌘.

## 4. Open a terminal (30 s · Pro)

⌘J opens the terminal panel at the bottom. The "+ Claude" button starts a tab where Claude is already running in your workspace folder. Type a question; Claude can see and act on your files. The same is true for the "+ Codex" and "+ Gemini" buttons if you have those CLIs installed.

## 5. Mark up what you're looking at (45 s)

⌘⌥P toggles Paint. Circle the thing you want to discuss, click the AI Assistant's 📎 button to attach the sketch to your next message, ask "what is wrong with this layout?". The screenshot lands in the conversation.

## 6. Record a voice memo (30 s · Pro)

Microphone icon in the bottom-right status bar → click → talk → click again. The recording lands in your workspace as a `.m4a` next to your files. The Voice Player at the bottom of the screen plays it back. If you want a transcript, ask the AI Assistant ("transcribe this memo").

## 7. Export the result (30 s)

⌘E opens the Export menu. Pick PDF for printing, Markdown for handing to another tool, Markdown (Smart) for the polished version, HTML for a single-file send. Each format remembers where you saved last time.
