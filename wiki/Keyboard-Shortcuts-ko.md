# 단축키

> 모든 단축키 한 페이지. *Settings → Keyboard* 에서 재바인드 가능.

## 파일 & 탭

| 단축키 | 동작 |
|---|---|
| ⌘O | 파일 열기… |
| ⌘⇧O | 폴더를 워크스페이스로 열기 |
| ⌘N | 새 윈도우 |
| ⌘T | 새 탭 (터미널 preset → 새 터미널) |
| ⌘W | 탭 닫기 |
| ⌘⇧T | 마지막 닫은 탭 다시 열기 |
| ⌘1..9 | 탭 N 점프 |
| ⌘\` | 다음 탭 (터미널 포커스 시 토글) |
| ⌘⇧\` | 이전 탭 |
| ⌘S | 저장 |
| ⌘⇧S | 다른 이름으로 저장… |
| ⌘P | 인쇄 |
| ⌘E | Export 메뉴 |
| ⌘, | Settings |

## 사이드바

| 단축키 | 동작 |
|---|---|
| ⌘⇧F | 워크스페이스 검색 |
| ⌘A | 전체 선택 (사이드바 안) |
| ⌫ | 선택 삭제 (2개 이상 확인) |
| ⌘⌥R | Finder 에서 표시 |
| ⌘+클릭 | 선택 추가 |
| ⇧+클릭 | 범위 선택 |
| 300 ms 길게 누르기 | 이름 변경 |

## 뷰어 / 에디터

| 단축키 | 동작 |
|---|---|
| ⌘F | 뷰어 안 찾기 |
| ⌘G | 다음 매치 |
| ⇧⌘G | 이전 매치 |
| ⌘B / ⌘I / ⌘U | 볼드 / 이탤릭 / 밑줄 (Markdown) |
| ⌘K | 링크 다이얼로그 (Markdown) |
| ⌘⌥0..6 | 헤딩 레벨 0..6 토글 |
| ⌘⇧V | 평문 paste |
| ⌘+ / ⌘− | 확대 / 축소 |
| ⌘0 | 줌 리셋 |

## 보기 모드

| 단축키 | 동작 |
|---|---|
| ⌘⇧D | Dual view 토글 |
| ⌘E (Pro) / ⌘⌥E (Easier) | Code view 토글 |
| ⌘⇧P | Paint 모드 토글 (Pro) |
| ⌘⌥P | Present 모드 토글 |

## 터미널

| 단축키 | 동작 |
|---|---|
| ⌘\` | 터미널 패널 토글 |
| ⌘T | 활성 preset 로 새 탭 |
| ⌘D | 활성 pane 좌우 split |
| ⌘⇧D | 활성 pane 상하 split |
| ⌘[ / ⌘] | pane 포커스 순환 |

## ChatPanel

| 단축키 | 동작 |
|---|---|
| ⌘L | ChatPanel 토글 |
| ⌘⏎ | 메시지 제출 |
| ⌘. | 스트리밍 응답 취소 |

## 영역 캡처 / 스크린샷

| 단축키 | 동작 |
|---|---|
| ⌘⌥R | 영역 선택 → 클립보드 |
| ⌘⌥⇧R | 영역 선택 → Paint 캔버스 |
| ⌘⌥⇧S | 현재 뷰어를 PNG 로 저장 |

## 음성 메모

| 단축키 | 동작 |
|---|---|
| (Settings 에서 바인드) | 녹음 시작 / 중지 |

## 메모

- 일부 단축키는 macOS 예약 조합과 충돌. 외부 키보드 클램쉘 설정에서도 동일 바인딩.
- `⌘M` (윈도우 최소화) 와 `⌘⇧M` 은 macOS 가 관리 — HTMLook 은 충돌 회피. dev 빌드에서만 `⌘⇧M` 사용.
