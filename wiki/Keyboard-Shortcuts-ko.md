# 단축키

> 네이티브 메뉴에 wire 된 모든 단축키. Apple Silicon Mac 기준 (⌘ = Cmd). macOS 상단 메뉴바가 정식 목록 — 여기는 거기 mirror.

## File

| 단축키 | 동작 |
|---|---|
| ⌘N | 새 텍스트 |
| ⌘⇧T | 템플릿에서 새로… |
| ⌘⇧N | 새 워크스페이스 마법사 (Pro) / 새 윈도우 (Easier) |
| ⌘⌥⇧N | 새 윈도우 (Pro) |
| ⌘O | 폴더 열기… |
| ⌘⇧O | 파일 열기… |
| ⌘S | 저장 |
| ⌘⇧S | 다른 이름으로 저장… |
| ⌘W | 탭 닫기 |
| ⌘P | 인쇄 |

## Edit

| 단축키 | 동작 |
|---|---|
| ⌘Z | 실행 취소 |
| ⌘⇧Z | 다시 실행 |
| ⌘⇧F | 워크스페이스에서 찾기 |
| ⌘L | 라인으로 이동… |

## View

| 단축키 | 동작 |
|---|---|
| ⌘1 | 미리보기 모드 |
| ⌘2 | 소스 모드 |
| ⌘3 | 분할 (미리보기 + 소스) |
| ⌘G | 갤러리 보기 |
| ⌘B | 사이드바 토글 |
| ⌘⌥P | Paint 토글 (Pro) |
| ⌘J | 터미널 토글 (Pro · macOS) |
| ⌘⌥⇧T | 선택을 터미널로 보내기 (Pro) |
| ⌘D | 터미널 split — 상/하 (Pro) |
| ⌘⇧D | 터미널 split — 좌/우 (Pro) |
| ⌘I | AI Assistant / ChatPanel 토글 (Pro) |

## 탭

| 단축키 | 동작 |
|---|---|
| ⌘⌥1 .. ⌘⌥9 | 탭 1..9 점프 |
| ⌘⌥→ | 다음 탭 |
| ⌘⌥← | 이전 탭 |
| ⌘⇧T (닫은 후) | 마지막 닫은 탭 다시 열기 |

## Markdown 에디터 안

| 단축키 | 동작 |
|---|---|
| ⌘B / ⌘I / ⌘U | 볼드 / 이탤릭 / 밑줄 |
| ⌘K | 링크 다이얼로그 (삽입 / 편집 / 제거) |
| ⌘⌥0 .. ⌘⌥6 | 헤딩 레벨 0..6 토글 |
| ⌘⇧V | 평문으로 붙여넣기 |

## 뷰어 안

| 단축키 | 동작 |
|---|---|
| ⌘F | 현재 파일 안에서 찾기 |
| ⌘G | 다음 매치 |
| ⇧⌘G | 이전 매치 |
| ⌘+ / ⌘− | 확대 / 축소 |
| ⌘0 | 줌 리셋 |

## ChatPanel (Pro)

| 단축키 | 동작 |
|---|---|
| ⌘I | ChatPanel 토글 |
| ⌘⏎ | 메시지 제출 |
| ⌘. | 스트리밍 응답 취소 |

## 사이드바

| 단축키 | 동작 |
|---|---|
| ⌘⇧F | 워크스페이스 검색 |
| ⌫ | 선택을 휴지통으로 (2개 이상 선택 시 확인) |
| ⌘+클릭 | 선택 추가 |
| ⇧+클릭 | 범위 선택 |
| 길게 누르기 (300 ms) | 이름 변경 |

## 윈도우

| 단축키 | 동작 |
|---|---|
| ⌘, | Settings |
| ⌘M | 윈도우 최소화 (macOS) |
| ⌘H | 윈도우 숨기기 (macOS) |
| ⌘Q | 앱 종료 |

## 참고

- 네이티브 메뉴바가 각 메뉴 항목 옆에 단축키 표시. 헷갈리면 거기 확인.
- `⌘L` 은 **라인 이동**, ChatPanel 아님 — ChatPanel 은 `⌘I`.
- `⌘\`` 는 빌트인 단축키 아님. macOS 에선 `⌘J` 로 터미널 토글.
- 일부 단축키가 Easier 와 swap (예: `⌘⇧N` 이 Easier 에선 *새 윈도우*, Pro 에선 *워크스페이스 마법사*).
