# Pro vs Easier

> Two builds for two audiences. Pick the right one for who uses it.

| | **HTMLook Pro** | **HTMLook Easier** |
|---|---|---|
| **Target user** | Power user, developer, prosumer with own AI keys | Novice / non-technical reader |
| **AI** | Bring Your Own Model — your OpenAI / DeepSeek / Mistral / Together / Groq / Cerebras / Ollama / Custom keys | Built-in subscription AI |
| **Terminal** | Full terminal with shell + AI presets (Claude / Codex / Gemini / Shell). The AI agents can see and act on your workspace files. | (none) |
| **Paint** | Sketch canvas on top of any file. 7 tools, undo / redo, PNG export, drag into chat. | (none) |
| **Voice memos** | Record AAC memos into the workspace, with optional transcripts you control. | (none) |
| **Extensions** | Add your own commands and renderers as plain Markdown files dropped into the workspace. | (none) |
| **Markdown editor** | Full WYSIWYG — autocorrect, ⌘K links, code-block language picker, paste sanitiser, custom print header/footer. | Read-only render |
| **Export** | PDF, Print, DOCX, Markdown, Markdown (Smart), HTML, plus dynamic extension-driven formats. | PDF, Print, HTML |
| **Auto-update** | Background download + restart prompt. Signed, verified. | Same. |
| **License** | Pro Solo $7/mo, 14-day free trial. Perpetual. | Subscription (covers the built-in AI). |
| **Privacy** | Nothing leaves the device except the AI calls you make yourself. | Subscription AI calls go through our backend. |

## When to choose Pro

Pick Pro when any of these is true:

- You already pay for an AI you want to use (Anthropic, OpenAI, Together, a local Ollama, …)
- You're comfortable with a terminal and want Claude / Codex / Gemini *inside* the app
- You want fine-grained control over what files leave your device
- You're a designer / writer / engineer / researcher / lawyer / PM who annotates a lot and wants Paint + voice memos baked in

## When Easier is the right call

- You don't want to think about API keys, monthly LLM bills, or which model
- You read way more than you write
- You want HTMLook to "just have AI" and you'll pay a monthly subscription for the convenience
- You're recommending the app to a family member or coworker who doesn't enjoy tools menus

## The honest tradeoff

Pro is more powerful and more honest about cost — you see the LLM bill, you pick the model, the app stays out of your wallet. Easier is friendlier on day one and the AI quality is consistent across users — you don't have to evaluate eight providers to get a good answer.

Both are fed by the same renderer, file watcher, and Markdown engine. A workspace opens in either build with identical fidelity. So if you ever want to switch you just install the other build and point it at the same folder — your files and notes travel with you.
