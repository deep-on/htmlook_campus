# 탭과 보기 모드

> 탭은 여러 파일을 동시에 열어두는 방식. 보기 모드는 한 탭의 화면 배치.

## 파일을 탭으로 열기

| 방법 | 트리거 |
|---|---|
| 사이드바 단일 클릭 | 현재 탭 교체 |
| 사이드바 더블 클릭 | 새 탭 |
| ⌘+클릭 | 백그라운드 탭으로 |
| Finder → 윈도우 드래그 | 새 탭 |
| 뷰어 안 `htmlook://` 링크 | 기존 탭 재사용 또는 새 탭 |
| MCP `htmlook_open_file` | 프로그램적 오픈 |
| ⌘O | OS open dialog |

## 탭 스트립 동작

- ⌘1..9 로 탭 1..9 점프
- ⌘W 활성 탭 닫기 (수정 시 확인)
- ⌘⇧T 마지막에 닫은 탭 다시 열기
- ⌘\` 다음, ⌘⇧\` 이전
- 가운데 클릭 닫기
- 가로 드래그로 순서 변경
- 탭을 strip 밖으로 드래그 → 새 윈도우로 tear-out
- ● 미저장 변경. 호버 시 × 로 변해 원클릭 닫기
- 터미널 탭은 letter mark (Cl/Cx/Gm/Sh). 출력 중에는 stream 애니메이션

### 탭 우클릭 메뉴

- *Pin / Unpin* — 핀 고정 탭은 ⌘W-all 에서 살아남음
- *다른 탭 닫기*, *오른쪽 닫기*, *모두 닫기*
- *Finder 에서 표시*
- *경로 복사*, *상대 경로 복사*
- *터미널에서 열기* (다음 비어있는 터미널 탭에서 파일 디렉토리로 chdir)

## 보기 모드 (활성 탭 단위)

### Single (기본)
중앙 영역을 하나의 뷰어가 채움.

### Dual (⌘⇧D)
중앙을 좌우 분할. 각 쪽이 다른 파일 또는 같은 파일의 다른 모드 (예: 왼쪽 `.md` 렌더, 오른쪽 raw 소스). 듀얼뷰 툴바의 **Sync scroll** 토글로 세로 위치 동기화.

### Code (Pro: ⌘E, Easier: ⌘⌥E)
파일 원본 소스를 렌더 옆에 표시. `.md`, `.html`, `.json`, `.csv`, `.svg` 지원. 소스 편집 시 라이브 반영.

### Paint (⌘⇧P, Pro 전용)
뷰어 위에 투명한 스케치 캔버스 오버레이. 그린 결과는 탭별 `.htmlook/sketches/<basename>.<timestamp>.png` 로 저장. [Paint](Paint-ko.md) 참조.

### Present (⌘⌥P)
chrome (사이드바 · 탭 · 상태바) 숨김. 프로젝션 / 녹화에 유용. ⎋ 로 종료.

## 저장 모델

Markdown / HTML 탭은 마지막 키 입력 1.5 s 후 자동 저장. 상태바에 보류 중 `●`, 커밋 후 `✓`. ⌘S 로 즉시 저장.

미저장 (Untitled) 탭은 자동 저장 안 함 — ⌘S 시 저장 위치 묻습니다.

## 탭 상태 복원

워크스페이스마다 마지막 세션의 탭을 복원. `.htmlook/session.json` 에 저장. 지워도 안전 — "지난번에 뭐 열었었나" 메모리만 잃을 뿐.
