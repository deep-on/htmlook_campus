LLM prompts for the kid-coding scenario — Korean ×5.
Drop any of these into your LLM (Claude, Codex, Llama via Ollama,
Gemini, GPT, etc.) with the HTMLook MCP attached. The two named
moves — explain code, next challenge — are [1] and [3].

Copy each prompt verbatim. The LLM is tutoring an elementary
school student writing her first Python turtle program.

────────────────────────────────────────────────────────────────────
[1] 이 부분이 뭐야? (the main move — explain a line)
────────────────────────────────────────────────────────────────────
좌측 pane 에 star.py 가 열려 있어. htmlook_cite 로 L7~L11 의
for 반복문 부분을 인용해 줘. 초등학교 5학년한테 설명한다고
생각하고, 한국어로 한 문장씩 친절하게 알려 줘. 어려운 단어는
쓰지 말고. 마지막에 "이제 한 번 바꿔볼까?" 같은 말로 끝내 줘.

────────────────────────────────────────────────────────────────────
[2] 첫 별을 그려 줘
────────────────────────────────────────────────────────────────────
star.py 를 실행하면 우측 pane 에 거북이가 별을 그려. 한 변씩
그려지는 모습을 글로 설명해 줘 — "위로 한 번, 오른쪽으로
한 번, 또 위로…" 하는 식으로. 5학년이 머릿속에 그림을
그릴 수 있게. 영어 단어는 쓰지 말고.

────────────────────────────────────────────────────────────────────
[3] 다음 챌린지 (the second main move — suggest the next step)
────────────────────────────────────────────────────────────────────
star.py 의 angle 값을 144 에서 72 로 바꿔 줘 (htmlook_apply_edit
left star.py). 우측 pane 이 다시 그리면 무슨 모양이 나올지
미리 한 줄로 예고해 줘. 그 다음에는 색깔을 magenta 에서
cyan 으로도 바꿔 보자.

────────────────────────────────────────────────────────────────────
[4] 색깔 4 개로 별을 그려 줘
────────────────────────────────────────────────────────────────────
star.py 를 4 번 호출해서 각각 magenta, cyan, yellow, 보라색
별을 그리도록 바꿔 줘. for 반복문 안에서 색깔이 바뀌도록
하면 더 멋있어. htmlook_apply_edit 로 한 번에 적용하고,
우측 pane 의 그림이 4 색 별로 바뀌는 걸 확인해 줘.

────────────────────────────────────────────────────────────────────
[5] 학습 일지에 오늘 배운 거 적어 줘
────────────────────────────────────────────────────────────────────
오른쪽 pane 을 docs/journal.md 로 바꿔 줘 (htmlook_set_pane
right docs/journal.md). 오늘 배운 거 세 줄로 정리해 줘:
"1) for 반복문이 뭔지, 2) angle 을 바꾸면 모양이 어떻게
변하는지, 3) 다음에 해 볼 건 뭔지." 5학년 말투로.
