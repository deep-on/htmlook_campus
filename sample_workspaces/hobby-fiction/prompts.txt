LLM prompts for the hobby-fiction scenario — Korean ×5.
Drop any of these into your LLM (Claude, Codex, Llama via Ollama,
Gemini, GPT, etc.) with the HTMLook MCP attached. The two named
moves — write the scene from outline, character consistency check
— are [1] and [3].

Copy each prompt verbatim. The LLM is co-writing chapter 1 of the
SF-mystery novella *화성의 마지막 편지*.

────────────────────────────────────────────────────────────────────
[1] outline 에서 한 장면 쓰기 (the main move)
────────────────────────────────────────────────────────────────────
좌측 pane 의 outline.md 에서 plot point "Mars colony 마지막
통신" 을 htmlook_cite 로 인용해 줘 (L24-L38). 그 장면을
chapter-1.md 의 끝에 이어서 써 줘. 대화 + 묘사 + 인물의
숨소리까지. 1500 단어 분량. 인물은 outline 의 character
card 그대로 — Dr. Han, Captain Ishara, 통신병 Mira. 이미 쓰여
있는 chapter 의 톤을 유지해 줘 (담담한 합니다체 narration,
짧은 문장 위주). 다 쓰면 htmlook_apply_edit 로 우측 pane 에
넣어 줘.

────────────────────────────────────────────────────────────────────
[2] 한 plot point 인용
────────────────────────────────────────────────────────────────────
htmlook_cite 로 outline.md 의 L24-L38 만 인용해 줘. 그
plot point 가 chapter 의 어느 단락과 연결되는지 한 줄로

────────────────────────────────────────────────────────────────────
[3] 인물 일관성 검사 (the second main move)
────────────────────────────────────────────────────────────────────
chapter-1.md 의 새로 쓴 마지막 장면에서 Dr. Han 의 대사
4 줄을 골라 줘. outline.md 의 Dr. Han character card
(L42-L58) 에 적힌 말투 — 합니다체, 자기 이름을 절대

검증해 줘. 어긋난 줄이 있으면 어떻게 고치면 되는지
한 문장으로 제안.

────────────────────────────────────────────────────────────────────
[4] 대화 다듬기
────────────────────────────────────────────────────────────────────
chapter-1.md 의 cite 한 마지막 장면에서 대화 부분만 골라
줘. "ㅎㅎ" "ㅋㅋ" 같은 입말 표시는 다 빼고, 호흡이 너무 긴
대사는 두 문장으로 쪼개 줘. htmlook_apply_edit 로 그
구간만 교체해 줘. 묘사는 건드리지 말고.

────────────────────────────────────────────────────────────────────
[5] 한 chapter 마무리 한 줄
────────────────────────────────────────────────────────────────────

넘어가는 hook 이 되어야 해 — 질문이거나, 미해결 사물이거나,
인물의 한 동작. outline.md 의 chapter 2 메모 (L80-L86) 와
모순되지 않게. 한 문장.
