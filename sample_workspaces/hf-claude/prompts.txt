[1] Simple request (main)
Build a 15-second product launch teaser for Acme Cloud. Near-black
canvas with a cyan(#06b6d4)→pink(#ec4899) gradient, kinetic
typography on three keywords ("ship faster", "cloud-native",
"today"), and an "A" monogram + Acme Cloud wordmark +
acmecloud.dev URL outro in the last 3 seconds. Four scenes
(0-3 / 3-8 / 8-12 / 12-15). Lint clean, then render.

[2] Refinement — make one keyword heavier
"ship faster" reads too quick. Increase letter stagger 0.045 → 0.09
and scale pulse 1.04 → 1.08 within the same 5-second window. Heavier
emphasis without changing duration.

[3] Outro punch
The outro mark settles too early. Drop the "A" monogram at 12.0 s
with a back.out, fade the wordmark at 12.4 s (power2.out), and pop
the URL last at 12.95 s with a 14 px y-rise. Each step heavier than
the previous.

[4] Domain variant
Same concept, fintech-SaaS variant. "Ledger Cloud", URL
ledgercloud.io, copy "settle faster", "audit-ready", "today".
Recolor cyan → emerald (#10b981), pink → amber (#f59e0b).

[5] 9:16 portrait variant
Re-author this composition for 1080×1920 portrait. Scale type to 80%
and stack the kinetic words vertically. Outro mark stays the same.
