LLM prompts for the teacher-record scenario — English ×5.
Drop any of these into your LLM (Claude, Codex, Llama via Ollama,
Gemini, GPT, etc.) with the HTMLook MCP attached. The two named
moves — group + draft, fill the quiet student — are [1] and [3].

Copy each prompt verbatim. The LLM is helping a Grade 3 homeroom
teacher (김민지, 햇살초 3-1반) write the term-end record book.

────────────────────────────────────────────────────────────────────
[1] Group + draft summaries (the main move)
────────────────────────────────────────────────────────────────────
You are helping me write the 학기말 학생 관찰 종합 의견 (term-end
student observation record) for class 3-1. daily-notes.md is on
the left pane (one raw note per line, tagged with 학생 이름) and
student-records.md is on the right (empty record book). Use
htmlook_cite to read both. Group the notes by 학생 이름 and draft
a 3 to 4 sentence term-end summary per student. Quote concrete
behaviors only — never invent a behavior the notes did not record.
Return as htmlook_apply_edit blocks I can land directly.

────────────────────────────────────────────────────────────────────
[2] Cite per-student anchors
────────────────────────────────────────────────────────────────────
For each student row you wrote, run htmlook_cite("left",
"<student> rows") and confirm in one sentence that the summary
only uses the dates+notes that anchor would return. If a sentence
relies on something outside the cite, mark it [unsupported].

────────────────────────────────────────────────────────────────────
[3] Fill the quiet student (the second main move)
────────────────────────────────────────────────────────────────────
서윤 is the quietest student in the class — only three notes for
her this term. htmlook_cite("left", "서윤 03-04, 03-18, 04-08")
and refine her row with four short concrete sentences plus a
one-line recommendation ("발표 기회를 짧게라도 정기적으로
마련해 주면 좋겠다"). Highlight the four sentences in the right
pane with <mark> so the parent's eye lands on cited evidence.
Return as a unified diff against student-records.md.

────────────────────────────────────────────────────────────────────
[4] Sweep for missed students
────────────────────────────────────────────────────────────────────
Scan daily-notes.md for any student name that does not yet have a
row in student-records.md. List each one with the count of notes
they have and the dates. Do not invent rows for students with zero
notes — flag them so the teacher can add observations manually.

────────────────────────────────────────────────────────────────────
[5] Add a printable footer + signing line
────────────────────────────────────────────────────────────────────
Open student-records.md (htmlook_set_pane right
student-records.md). Add a printable footer with school name,
class, term, 담임 name, date, and a signing line so the printed
record can be filed in the cabinet. Keep the calm observational
voice. Return as a diff.
