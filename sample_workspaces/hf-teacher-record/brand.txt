Class shown:    3학년 1반 · 24명 · 담임 김민지

Palette
  --paper       #ffffff   white record-book paper
  --beige-bg    #fffbe8   notebook beige canvas
  --beige       #fef3c7   notebook stripe (header / numbers)
  --highlight   #fde68a   형광 노랑 (cited mark)
  --highlight-2 #fbbf24   stamp / accent (term seal)
  --ink         #1c1917   slate ink (titles, body)

Each student gets one band color so the eye can follow them across
the four scenes — class roll → raw notes → grouping → record book.

Typography
  Display       Inter / Pretendard, 700-800 weight, tight tracking
  Body          -apple-system / Inter, 400-600
  Mono / notes  ui-monospace / SF Mono — used for raw daily notes
                and date stamps
  Highlight     <mark> with --highlight background, padding 0 4px

Motion language
  Class-roll cards stagger in left-to-right; the term stamp lands
  with a back-out at +4°. Daily-note rows reveal one date stripe
  at a time, then notes within. Grouping scene springs each
  column up sequentially (민준 → 서윤 → 지후), and items inside
  stagger from the top. Record-book rows slide left, with the
  highlighted lead sentence painting in last so the eye lands on
  the cited evidence.

Copy guardrails
  Header line:        "학생 관찰 기록부 · 학기말 종합 의견" + class
                      + term + 담임 name
  Voice:              calm, classroom-y, observational. Past or
                      present continuous tense. Concrete actions
                      ("도서 추천 발표를 준비했다"), not verdicts
                      ("우수한 학생이다"). One short
                      recommendation per quiet student is OK.
  Cite mark:          <mark> only on lead sentences that came
                      directly from a daily note — never marketing.
  Watermark:          "학기말 종합 의견 · 2026 1학기"
  Avoid:              "smart", "AI-powered", "magic", "blazing".
                      Also: never invent a behavior the daily
                      notes did not record.
