달팽이의 학교 (Snail School) — fictional Korean picture book for the picture-book scenario.

Working title:   달팽이의 학교
Author:          글 ◯◯◯ · 그림 ◯◯◯
Format:          16:9 spread (펼침면) — left page text, right page illustration
One-liner:       Verse + page rhythm, in one view.

Palette
  --paper       #fef3c7   ivory book paper, warm
  --ink         #3f3f46   handwriting / stanza text
  --mint        #a7f3d0   illustration accent (snail trail, leaves)
  --pink        #fbcfe8   illustration accent (dandelion blossom)
  --soft        #d4d4aa   muted paper edge / margin annotations
  --rule        #e7e0c4   page-fold rule, soft grid

Typography
  Display       "Nanum Pen Script" / "Gaegu" / handwriting Korean serif,
                400-500 weight, generous leading
  Mono (cite,
   anchors)    SF Mono / Menlo / ui-monospace, 2-3px tracking
  Body         "Iowan Old Style" / "Charter" / serif for English glosses

Motion language
  Watercolor pastel reveal. Stanza lines land one row at a time, never
  per-character (this is a poem, not a kinetic ad). Right-page illustration
  fades in as a soft watercolor wash — snail silhouette + dandelion bloom
  in mint and pink. Page-turn at the very end is a single soft fold, not
  a flip cycle. Background is paper, not paper-with-grain animation —
  keep the canvas still and breathing.

Verse guardrails
  Three stanza lines:   "느릿느릿 달팽이"
                        "학교 가는 길에"   →   "학교 가는 길 위에"
                        "민들레 한 송이"
  Caption (right page): "갓 핀 민들레"   →   "갓 핀 민들레꽃"
  Wordmark close:       "달팽이의 학교 · 출간 예정"
  Voice:                soft, child-facing, present tense. No exclamation
                        marks. No emoji. No marketing copy.
  Avoid:                "magical", "AI-generated", "auto-rhymed" — this
                        is a working picture-book draft, not a launch.
  Syllable counts:      keep each line at four Korean syllables when
                        possible (children's reading rhythm). Tweaks
                        that change syllable count must be flagged in
                        the cite anchor.
