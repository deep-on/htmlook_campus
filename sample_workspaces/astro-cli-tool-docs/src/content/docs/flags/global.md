| flag | type | default | description |
|------|------|---------|-------------|
| `--config` | path | `acme.yaml` | config file location |
| `--verbose` | bool | false | extra log output |
| `--format` | enum | text | text · json |
| `--no-color` | bool | false | disable color |
| `--help` | bool | — | show help |
| `--version` | bool | — | print version + exit |
