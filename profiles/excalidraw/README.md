# Excalidraw Profile

손그림 화이트보드 [Excalidraw](https://excalidraw.com) HTMLook Profile.

## 시작

- **Web**: https://excalidraw.com → "Open" → `seed/diagrams/architecture.excalidraw`
- **VS Code**: Excalidraw extension 설치 후 폴더 열기
- **Desktop**: https://github.com/excalidraw/excalidraw-desktop

## HTMLook 활용

- **Region cite ⭐**: 다이어그램 영역 캡처 → AI 에게 글 설명 / 후보 라벨.
- **Pane pair**: 좌 `.excalidraw` JSON ↔ 우 SVG/PNG export 미리보기.
- **Multi-target**: 다이어그램 + 본문 md 동시 수정.
