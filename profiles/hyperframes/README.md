# HyperFrames Profile

kinetic typography 영상 합성 [HyperFrames](https://github.com/deep-on/hyperframes) HTMLook Profile.

## 시작

```bash
npm install hyperframes
mkdir renders
npx hyperframes render
```

## HTMLook 활용

- **Live-cue ⭐**: 영상 timestamp ↔ scene 라인 양방향.
- **Pane pair**: scene.md ↔ rendered mp4.
- **Multi-target apply_edit**: 여러 scene 동시 수정.

## 외부 의존

- Node 18+, ffmpeg, Kokoro TTS (Python).
