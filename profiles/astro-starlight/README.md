# Astro Starlight Profile

모던 docs 사이트 [Astro Starlight](https://starlight.astro.build) HTMLook Profile.

## 시작

```bash
pnpm install
pnpm dev   # http://localhost:4321
```

## HTMLook 활용

- **Pane pair**: 좌 `.md`/`.mdx` ↔ 우 live docs.
- **Multi-target apply_edit ⭐**: 사이트 전체 용어 / 링크 일괄 수정.
- **Region cite**: 코드 블록 / 다이어그램 영역 → AI 에게 보강.

## 외부 의존

- Node 18+, pnpm/npm
