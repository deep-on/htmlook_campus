# 수학 숙제 풀이 노트북

## 문제 1 풀이

> 직각삼각형 ABC, 직각변 5 와 12, 빗변 c.
> 출처: `homework.md` 문제 1.

**Step 1.** 피타고라스 정리를 적용한다.

$$a^2 + b^2 = c^2$$

**Step 2.** 알고 있는 값을 대입한다 ($a = 5, b = 12$).

$$5^2 + 12^2 = c^2$$
$$25 + 144 = 169$$

**Step 3.** 양변에 제곱근을 취해 $c$ 를 구한다.

$$c = \sqrt{169} = 13$$

**답.** 빗변의 길이 $c = 13$.

> 한 줄 reminder: *직각삼각형이라면 두 직각변 제곱의 합이 빗변의 제곱.*

## 비슷한 연습 문제

세 문제 모두 $a^2 + b^2 = c^2$ 를 그대로 적용하면 된다 — 숫자만 바꾼다.

1. 직각변 $3, 4$ → $c = ?$ (답: $5$, 가장 유명한 3-4-5 직각삼각형)
2. 직각변 $8, 15$ → $c = ?$ (답: $17$)
3. 직각변 $9, 40$ → $c = ?$ (답: $41$)

각 문제마다 위의 Step 1-3 양식을 똑같이 따라 적어 보자. 답은 위에 표시된
정수가 나온다 (모두 *피타고라스 수* 묶음).
