[14:02] **Editor** — Why do you push back when people say models are everything?

[14:08] **Park** — Because every shop has the same model now.

[14:14] **Park** — *The model is a commodity. The workflow around it is the product. That's the moat.*

[14:34] **Park** — The team that ships the cleanest cite-and-edit loop wins. That part is unfakeable.

[14:48] **Editor** — So when junior PMs ask which model to pick —

[14:52] **Park** — I tell them: the model picks itself. Pick the seams.

[15:14] **Park** — File watchers, anchors that survive merges, an undo stack you actually trust.
